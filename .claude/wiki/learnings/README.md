# Learnings

Pelajaran yang didapat dari project nyata — bukan teori, tapi pengalaman aktual yang layak diingat untuk project berikutnya.

**Beda dengan:**
- `patterns/` → pola solusi yang terbukti
- `gotchas/` → jebakan yang harus dihindari
- `errors/` → kasus error spesifik
- `learnings/` → insight umum / refleksi (mis. "estimasi feature X selalu meleset 2x karena Y")

> Folder ini masih kosong. Tambahkan entry setelah retrospektif project atau saat sadar pola berulang.

## Kapan Dibaca

- Mulai project baru dengan tipe / stack mirip
- Estimasi effort untuk task yang mirip historis
