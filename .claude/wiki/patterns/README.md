# Patterns

Resep / pattern teknis yang sudah dipakai dan terbukti. Setiap entry: problem, solusi, contoh kode, dan catatan trade-off.

**Format entry:** problem singkat → solusi (snippet/diagram) → kapan dipakai → trade-off → referensi project asal.

> Folder ini masih kosong. Tambahkan entry saat sebuah solusi sudah dipakai di ≥2 project dan layak distandarkan.

## Kapan Dibaca

- Implementasi feature mirip dengan project sebelumnya
- Sebelum memutuskan pendekatan baru — cek dulu pattern yang sudah ada
