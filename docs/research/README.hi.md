## कपन पकाई फोल्डर इनि?

✅ **हां, सिम्पन दी सिनी:**
- कॉम्परसी लाइब्रेरी/फ्रेमवर्क/टूल
- एक्सप्लोरासी एपीआई व्यवहार
- प्रदर्शन/लागत की जांच
- स्पाइक रिसेट सेबेलम एंबिल केपुटुसन
- हासिल दारी `@research-assistant`

❌ **बुकन दी सिनी:**
- केपुटुसन अंतिम यांग सुदाह अपरिवर्तनीय → `.claude/wiki/decisions/` (एडीआर)
- पुन: प्रयोज्य पैटर्न → `.claude/wiki/patterns/`
- सबक सीखा यांग सदाबहार → `.claude/wiki/learnings/`
- योजना कार्यान्वयन → `docs/plans/`
- सत्र संदर्भ क्षणिक → `.claude/memory/`

## स्थिति प्रवाह

| स्थिति | आरती | कार्रवाई |
|---|---|---|
| `draft` | बारू दिबुअत, मसीह कोसोंग/सेतेंगा | लंजुट आईएसआई |
| `active` | रिसेट जालान | अद्यतन निष्कर्ष |
| `finalized` | सेलेसाई, एडा रेकोमेंडासी | बुअत एडीआर / योजना तुरुनान |
| `superseded` | सुदह दिगंतिकन रिसेट/केपुटुसन बारू | `superseded_by` सेट करें |
| `archived` | यह प्रासंगिक है | पिंडाह के `_archive/` |

## फ्रंटमैटर वाजिब

देखें [docs/templates/research.md](../templates/research.md). फ़ील्ड पेंटिंग:
- `id` - नामा फ़ाइल से मिलान करें
- `status` — `grep` के माध्यम से फ़िल्टर खोलें
- `related` - एडीआर/योजना के लिए क्रॉस-रेफरी
- `promoted_to` — लिंक के विकि प्रविष्टि कलौ इनसाइट डि-एक्सट्रैक्ट
