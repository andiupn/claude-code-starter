文件夹 **riset eksplorasi yang time-bound**。 Tujuannya：simpan 散文和 hasil raiset sebagai artifak repo 琼脂：
- Bisa dirujuk lagi saat keputusan terkait dipertanyakan
- Tidak hilang saat session AI direset
- Bisa dipromosikan ke `.claude/wiki/` saat Insight-nya terbukti 常青树

## Kapan Pakai 文件夹 Ini？

✅ **是的，简单地说：**
- Komparasi 库/框架/工具
- Eksplorasi API 行为
- 调查性能/成本
- Spike raiset sebelum ambil keputusan
- 哈西尔·达里 `@research-assistant`

❌ **不可以：**
- Keputusan 最终 yang sudah 不可变 → `.claude/wiki/decisions/` (ADR)
- 模式可重用 → `.claude/wiki/patterns/`
- 阳常青的经验教训 → `.claude/wiki/learnings/`
- 计划实施 → `docs/plans/`
- 会话上下文短暂 → `.claude/memory/`
